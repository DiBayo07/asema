Как запустить проект

1. Установи Python 3.10 или выше.

2. Открой терминал в папке backend и установи зависимости:

   pip install -r requirements.txt

3. В папке backend находится файл .env. В нём хранятся логин и пароль от админки.
   По умолчанию: admin / 1234.
   Если хочешь сменить — отредактируй этот файл.
   Важно: .env не загружается на GitHub, поэтому пароли не утекут.

4. Запусти сервер из папки backend:

   python server.py

5. Открой браузер и перейди по адресу:

   http://localhost:5000

   Это главная страница сайта.

6. Админка доступна по адресу:

   http://localhost:5000/admin/login

   Войди с логином и паролем из .env.

Что есть в проекте

- Главная страница с секциями, тренерами, расписанием и формой записи.
- Админ-панель для просмотра заявок, управления тренерами и статистики.
- Защита от подбора пароля: после 5 неверных попыток IP блокируется
  (сначала на 5 минут, потом дольше).
